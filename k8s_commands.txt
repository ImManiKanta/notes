-> kubectl api-resources

namespace
=========
-> kubectl get namespace
-> kubectl get namespace or kubectl get ns
-> kubectl create namespace roboshop (through cli we can use this) or u can write yaml file
-> kubectl apply -f 01-namespace.yaml 
-> kubectl delete -f 01-namespace.yaml-> to execute the changes
-> kubectl describe ns <namespace-name>
-> kubectl delete namespace roboshop 

pod
===
-> kubectl apply -f 02-pod.yaml
-> kubectl get pods -o wide -> it will fetch from default namespace
-> kubectl get pods -o wide -n <namespace-name>
